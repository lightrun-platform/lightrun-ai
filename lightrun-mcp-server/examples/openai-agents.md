# OpenAI Agents example

Use the Lightrun MCP server URL as a remote MCP endpoint:

```text
https://app.lightrun.com/mcp
```

For single-tenant and self-hosted deployments, replace the URL with your organization's Lightrun MCP endpoint.

Authentication should use the method supported by your Lightrun environment, such as OAuth or API key Bearer authentication.

See the official setup guide:

https://docs.lightrun.com/mcp/mcp-quickstart/
